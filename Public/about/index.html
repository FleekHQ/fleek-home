We Believe in an Open, Trustless,
and Permissionless Web.

The team at Fleek is helping to build the tools necessary to make the Open Web available to anyone, and easy to build on. We want to enable the transition to the new internet: Open, trustless, permissionless, borderless, and user controlled.

The Internet’s Broken,

In Many, Many Ways.

The Old Web has many design flaws built-into its architecture that make it risk-prone and fragile. Users are not in control, networks are siloed, and the infrastructure and protocols are centralized on a handful of actors, putting it both at risk of failure, and abuse of power.

It’s Time to Rebuild
the Web’s Base Layer.

To transition to a new and fair web, we need to question the current web stack and rebuild from the ground up using Open Web protocols that make a decentralized, trustless, and permissionless infrastructure native to the web. This stack is ever shifting, and open-source.

What’s Fleek’s Role on this Path?

We are building an Open Web services interface and protocol layer to make the base layer of web services of the new internet (storage, hosting, billing etc.) accesible to anyone. We are exposing an open infrastructure, abstracting and simplifying protocols, open-sourcing tools, and opening up the ecosystem so that it is as easy to build on as the current web is.

Open Sourcing Tools,

Building on these protocols isn’t enough. We must make it as easy to work with these protocols as the current web is today. We are creating open source libraries like the Space SDK to achieve that and help anyone create sites & apps in the Open Web.
